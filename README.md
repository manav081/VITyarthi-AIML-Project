# VITyarthi-AIML-Project
A machine learning-based application designed to automatically detect and classify plant diseases from leaf images, helping farmers and botanists identify crop health issues early.
                                          PLANT DISEASES RECOGNITION 
 
 
INTRODUCTION : 
 
PLANTS CAN GET MANY DISEASES , AND FARMERS FACES A LOT OF PROBLEM  BECAUSES THEY CANNOT IDENTIFY THE DISEASES. 
 
THIS PROJECT USES DEEP LEARNING (CNN model) TO DETECT DISEASES FROM PLANT LEAF IMAGES AUTOMATICALLY. 
 
UPLOAD A PICTURE OF A LEAF . THE MODEL ANALIZE THE  PICTURE AND THEN TELL YOU THE LEAF  IS HEALTHY OR DISEASED. 
    
 
 
PROJECT AIM : 
 
TO BUILD A  MODEL THAT CAN RECOGNIZE PLANT DISEASES AUTOMATICALLY. IT HELP FARMERS AND AGRICULTURE  WORKERS TO IDENTIFY  THE PROBLEMS EARLY. 
TO REDUCE THE LOSS OF CROPS BY PROVIDING QUICK AND ACCURATE DISEASE DETECTION.

HOW MODEL WORKS:  
 
1.	INPUT - IMAGE OF A LEAF  
2.	PREPROCESSING -  RESIZE , NORMALIZE , CLEAN  
3.	MODEL - A CONVOLUTIONAL NEUTRAL NETWORK (CNN) LEARN FEATURE 
4.	OUTPUT - DISEASES NAME OR “HEALTHY LEAF “ 
  
 
RUN THE PROJECT :  
 
1.	CLONE THE REPOSITORY  
Git clone https://github.com/manav081/VITyarthi-AIML-Project 
 
2.	INSTALL REQUIRED  LIBRARIES     pip install -r reequirments.txt 
 
3.	TRAIN THE MODEL :    python src/train.py 
 
4.	PREDICT DISEASES FROM A LEAF IMAGE :  
 Python src/predict.py –image leaf.jpg
